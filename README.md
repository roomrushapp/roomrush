# RoomRush

RoomRush is a sublet marketplace for Munich helping students and young professionals find rooms faster.

🌐 Try it here: https://getroomrush.de

---

## What it does

- Browse available sublet listings in Munich  
- Filter by price, location, and availability  
- View listing details and contact options  
- Share listings easily  
- Tracks user interactions (contacts, shares)

---

## Why RoomRush

Finding short-term housing in Munich is chaotic:
- scattered across WhatsApp, Facebook, WG-Gesucht  
- no structured discovery  
- time-sensitive listings disappear fast  

RoomRush aims to make this faster and more transparent.
